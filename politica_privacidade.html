Política de Privacidade

1. Introdução

Esta política de privacidade descreve como a aplicação Realcloset recolhe, utiliza e protege as informações dos utilizadores. Esta aplicação foi desenvolvida para fins académicos, como parte do trabalho da unidade curricular de Desenvolvimento de Aplicações Móveis, do curso de Engenharia Informática, no Instituto Politécnico de Tomar

2. Informações Recolhidas

A aplicação pode recolher e processar as seguintes informações:

  • Informações Pessoais: Se o utilizador criar uma conta ou introduzir dados na aplicação, como nome de utilizador e senha.
  • Interações com a API: A aplicação comunica com uma API personalizada, alojada no Azure, para guardar e consultar informações.

3. Utilização das Informações

As informações recolhidas são utilizadas exclusivamente para o funcionamento da aplicação, incluindo:

  • Gestão de utilizadores e autenticação.
  • Interações com a base de dados associada à aplicação.
  • Validação de dados introduzidos.

Esta aplicação não recolhe informações com fins comerciais.

4. Partilha de Informações

Os dados dos utilizadores não são partilhados com terceiros. Apenas o grupo responsável pelo projeto e os docentes (se requisitado) têm acesso às informações durante o processo de avaliação académica.

5. Segurança

As informações são protegidas por medidas de segurança padrão, incluindo o uso de serviços da Microsoft Azure, que garante conformidade com padrões de segurança reconhecidos.

6. Direitos dos Utilizadores

Os utilizadores podem solicitar acesso ou remoção dos seus dados através de contacto com o grupo responsável pela aplicação.

7. Alterações

Dado o caráter académico do projeto, esta política poderá ser alterada para refletir ajustes no desenvolvimento da aplicação.

8. Contactos

Em caso de dúvidas, o utilizador pode contactar o grupo responsável pelo projeto.
